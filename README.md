The Clever Llamas Llamaverse is a data-rich fictional world and MarkLogic sample dataset built for demos, workshops, and hands-on learning.

Its purpose is to provide realistic, interconnected sample data so teams can learn and showcase MarkLogic search, semantic triples, ontology projection, content services, and geospatial query patterns in one coherent domain.

Meta description: Clever Llamas Llamaverse is a MarkLogic sample dataset and demo universe featuring llama people, places, schools, families, farms, and grazing zones for exploring document modeling, semantic ontology triples, geospatial search, and API-driven application development.

Details can be found here:[https://cleverllamas.com/articles/llamaverse](https://cleverllamas.com/articles/llamaverse/)
